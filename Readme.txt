使用说明
	打开游戏进入主界面，上下键设置胜利所需进球数，左右键设置AI难度，回车开始游戏；
	游戏中，鼠标控制曲棍位置，上下键伸缩镜头，左右键调整视角，P键暂停；
	暂停界面中，回车继续游戏，R键回到主界面；
	每次进球后，如果双方未胜利，进入进球提示界面，否则进入结束界面；
	进球提示界面中，回车继续下一球的比赛，R键回到主界面；
	结束界面中，回车继续新的一局比赛，R键回到主界面；
	视角调整和镜头伸缩在暂停、进球、结束界面同样适用；
	每一次游戏开始前，会有3s的倒计时。
AI说明
	Easy AI：在一条线段上来回移动，无视冰球位置；
	Normal AI：根据冰球位置，进行简单的阻挡；
	Crazy AI：根据冰球的位置和移动方向，进行阻挡，较难战胜。
附加实现

	4. 实现计分板；
